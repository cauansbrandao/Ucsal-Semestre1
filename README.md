# 🎓 Engenharia de Software - 1º Semestre (UCSAL)

Este repositório foi criado para armazenar e organizar todos os projetos e exercícios desenvolvidos durante o meu primeiro semestre na **Universidade Católica de Salvador (UCSAL)**.

## 📌 Objetivo
O foco principal aqui é documentar o meu aprendizado inicial em lógica de programação e a sintaxe básica da linguagem **Java**. Este repositório serve como uma "linha do tempo", onde pretendo manter os códigos originais para posterior análise e refatoração.

## 🚀 Tecnologias Utilizadas
* **Java:** Desenvolvimento de algoritmos e lógica de programação.

## 📂 Estrutura do Repositório
* `lista_01_Java_E_S/`: Exercícios iniciais focados em entrada/saída de dados e cálculos matemáticos.
* `lista_02_Java_condicional/` : Exercícios focados em estruturas de decisão (IF/ELSE) e lógica condicional.
* `lista_03_condicional/` : Exercícios de fixação em estruturas condicionais e introdução ao Switch Case.
* `lista_04_repeticao/` : Exercícios focados em laços de repetição (while, do-while e for).
* `lista_05_repeticao/` : Exercícios de fixação em laços de repetição e processamento de fluxos de dados.
* `lista_06_modularizacao/` : Exercícios focados na criação de métodos e passagem de parâmetros.
* `lista_07_strings/` : Exercícios focados na ultilização de strings.

## 📈 Evolução e Refatoração
Como agora estou no **2º semestre**, meu objetivo é revisar estes códigos periodicamente para aplicar novos conceitos de:
- **Clean Code** (Código Limpo)
- **Modularização**
- **Tratamento de Exceções**
